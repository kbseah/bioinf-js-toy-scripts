Bioinformatics Tools

Here are some simple scripts for manipulating bioinformatics data. They are implemented as webpage forms, so input can be copied and pasted in, and the output can be copied and pasted into your own files. How big can the input be? I don't know - but remember that this is Javascript we're talking about here.

The types of tools here are eclectic. They're mostly little things that I've found myself needing to do but couldn't find implemented somewhere convenient. I'm not sure who will use this - if you find yourself needing to match left and right brackets in a dot-bracket string chances are you already know how to do it - but perhaps this will save someone a half-hour or so of troubleshooting their own scripts.

The usual cautions apply: GIGO, and caveat emptor. They've worked for me, but perhaps you'll find some error that I've overlooked. Let me know if that happens.

List of tools

  • charrun looks for character 'runs' (same character repeated in adjacent positions) for a given character (user-specified) in strings of text.
  • bracketmatch matches up complementary left- and right-brackets in dot-bracket representations of RNA secondary structure, and spits out a list of paired base positions.
  • GCcalc finds the GC% for each sequence in a FASTA-formatted input.
  • replaceme takes an input text, a word list A, and a list of replacement words B, and swaps out the first instances of words in A with their corresponding aliases in B.
  • codoncount counts the number of occurrences of each of the 64 possible codons in each sequence of a FASTA-formatted input. Useful data for studying codon bias.
Full instructions for using the tools are on the separate pages. Suggestions welcome, but no tech support is available.
